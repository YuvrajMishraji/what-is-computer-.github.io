What is Computer?

• Definition:-A computer is an electronic machine that calculates arithmetic and logical operations.

• How Computer Works?:-
Firstly, it takes input data through input devices and processed them by CPU (Central Processing Unit).Finally, it gives result as output data through output devices.

• Parts Of Computer:-
There are four main parts of computer:-

1.CPU
2.Monitor
3.Keyboard
4.Mouse

Now we describe all four parts one by one:-

1.CPU:-CPU is the central processing unit which takes input data through input device and processed them and give us result as output. CPU has three parts:-
i)MU (Memory Unit)
ii)ALU (Arithmetic and Logic unit)
iii)CU (Control Unit)

2.Monitor:-Monitors are type of LED displays which uses an array of light-emitting diodes as pixels. It shows output as video, images etc.

3.Keyboard:-Keyboard is a type of device which use to type letters, symbols etc. in computer as input. It is an input device. Keyboard is of two types, Wired and Wireless Keyboard. Wired keyboards are connected with wires whereas wireless keyboards are connected wirelessly through infrared (IR) or Bluetooth technology.

4.Mouse:-A computer mouse is a type of pointing device (hand control) which allows a smooth scroll. Mouse is of two types, Wired and Wireless Mouse. Wired mouse are connected with wires whereas wireless mouse are connected wirelessly through infrared (IR) or Bluetooth technology.

•Some Input Device:-
Keyboard, Mouse, Light Pen, Track Ball, Scanner, Digitizer, Joystick etc.

•Some Output Device:-
Monitor, Printer, Plotter, Projector, Speakers, Headphones, Sound Card, Video Card etc.
